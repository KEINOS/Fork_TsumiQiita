# インストール方法

## Windows7^

1. gitをインストール (<https://git-scm.com/downloads>)
1. npmをインストール (<https://nodejs.org/ja/>)
2. Node.js用のコマンドプロンプトを開く
3. `$ git clone https://github.com/hidao80/TsumiQiita.git`
4. `$ cd TsumiQiita\src`
5. `npm install`
6. `npm run build-win`
