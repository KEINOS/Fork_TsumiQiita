# インストール方法

## Windows7^

1. npmをインストール（https://nodejs.org/ja/）
2. Node.js用のコマンドプロンプトを開く
3. `$ git clone https://github.com/hidao80/TsumiQiita.git`
4. `$ cd TsumiQiita\src`
5. `npm run install`
6. `npm run build-win`

## macOS 10.13^
